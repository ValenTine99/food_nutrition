# Мы - то, что мы едим!  

Это был мой учебный проект, но он достаточно сложный и интересный. Тут можно посмотреть на методы парсинга сайта, у которого нет API, посмотреть какие-то модельки, которые предсказывают рейтинг рецепта, а еще найти кучу рецептов по списку определенных ингредиентов.  

Когда-то случится транслейт на русский язык, а потом и супер удобное приложение, но это пока в процессе.

### Исходные данные  

Здесь использовались данные из репозитория классного человека, который решил вообще собрать 20 миллионов рецептов ... - https://www.kaggle.com/datasets/hugodarwood/epirecipes  

### Парсинг ссылок рецептов  

К сожалению, для каждого рецепта ссылка не была сохранена, поэтому пришлось придумать что-то, чтобы не умер мой комп, но при этом смог распарсить 20 миллионов ссылок, даже страшно писать это. Парсинг очень не замысловатый, можно посмотреть в файле **src/pars_url.py**.  
Или сразу взять уже файлик со всеми ссылками для работы - **data/url_recipes.csv**.
